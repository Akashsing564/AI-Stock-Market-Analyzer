# AI Stock Market Analyzer

## 🚀 Overview
AI Stock Market Analyzer is an intelligent stock analysis platform for the Indian Stock Market (NSE/BSE).

## Features
- 📈 Live Stock Data
- 🤖 AI Buy / Sell / Hold Signals
- 📊 RSI, MACD, EMA & Bollinger Bands
- 📰 News Sentiment Analysis
- 💼 Portfolio Tracker
- 🔔 Telegram Alerts
- 📉 Backtesting
- 🌐 Streamlit Dashboard

## Tech Stack
- Python
- Streamlit
- Pandas
- yfinance
- pandas-ta
- Plotly
- OpenAI API
